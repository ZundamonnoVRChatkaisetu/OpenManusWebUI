OpenManus WebUI ツール設定

GitHub設定

GitHub APIと連携してリポジトリの検索、ファイル操作、イシュー管理などを行います。

GitHubのPersonal Access Tokenを入力してください。リポジトリアクセス権限が必要です。
Web検索設定

Brave Search APIを利用してWeb検索を行います。最新の情報や事実確認に役立ちます。

Brave Search APIからAPIキーを取得してください。
